# Tabuada Com Estrutura De Repetição

programa {
	funcao inicio() {
		inteiro contador,limite,resultado
		
		contador = 0
		limite = 10
		
		faca {
		    resultado = 9* contador
		    escreva("9 x " + contador + "=" + resultado + "\n")
		    contador++
	}
	enquanto (contador<=limite)
}

}

9 x 0=0
9 x 1=9
9 x 2=18
9 x 3=27
9 x 4=36
9 x 5=45
9 x 6=54
9 x 7=63
9 x 8=72
9 x 9=81
9 x 10=90

Programa finalizado.
